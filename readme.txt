Sports News Widget for Sports Blogs Content by Blitzcorner.com

== Description ==

Blitzcorner Sports news widget is a great solution for sports blogs that want some extra content. The posts on the widget comes from the blogs in Blitzcorner network.
Install this widget give your user the window to a lot of other sports blogs from around the web.

== Installation ==

1. Unzip sports-news-widget.zip into /sports-news-widget folder.
2. /sports-news-widget folder should contain 3 files (sports_news_widget.php, readme.txt and screenshot-1.png)
3. Upload /sports-news-widget folder to your /wp-content/plugins/ directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Add the widget to your sidebar (Navigate to Appearance menu > Widgets and drag the widget to the prefered place on your sidebar)
